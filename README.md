# UrsaMajor

## Project Layout

In this Unity project, the following folders has the following roles:

- Required by Unity:
	- Assets
		- A folder required by Unity.  Contains assets used for the game.
	- Packages
	- ProjectSettings
  
- External Assets (these require download through package manager)
  - Nebula Skyboxes
